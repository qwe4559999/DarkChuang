# 更新日志

本文档记录了化学问答机器人项目的所有重要更改。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [2.2.1] - 2025-12-23

### 修复
- 🐛 **数学公式渲染**
  - 修复了 LaTeX 公式（如 `$$\text{不饱和度} = ...$$`）无法正确渲染的问题。
  - 引入 `KaTeX` 库，支持复杂的数学公式和化学方程式（如 `\ce{H2O}`）的渲染。
  - 修复了行内公式和块级公式的定界符解析问题，支持 `$$...$$`, `\[...\]`, `$ ... $`, `\(...\)` 等多种格式。

## [2.2.0] - 2025-12-23

### 新增
- 🧊 **3D 分子可视化**
  - 新增 `generate_3d_structure` 后端工具，基于 RDKit 生成 SDF 格式的 3D 分子数据。
  - 前端集成 `3dmol.js`，支持在聊天界面直接查看交互式 3D 分子模型。
  - 支持 Stick, Sphere, Line 等多种显示模式切换。
- 🔍 **全屏预览功能**
  - 新增 `MediaModal` 组件，支持图片和 3D 模型的全屏放大预览。
  - 优化了 Markdown 中图片的渲染，自动添加放大图标。
- 💅 **UI 交互优化**
  - 重构了底部聊天输入栏 (`ChatInput`)，采用悬浮胶囊设计，更加现代美观。
  - 优化了文件上传和发送按钮的布局与交互反馈。
  - 隐藏了聊天界面中内部工具调用的 JSON 指令，提升阅读体验。

### 改进
- ⚡ **性能与稳定性**
  - 优化了 `chemistry_service` 的分子解析逻辑，增加了常用名缓存和启发式检查，显著减少了 RDKit 错误日志和网络请求卡顿。
  - 修复了数据库会话管理问题，解决了多轮工具调用时 `Instance has been deleted` 的错误。
- 💾 **数据持久化**
  - 升级了数据库 Schema，新增 `data` 字段用于存储结构化消息数据（如 3D 模型 SDF）。
  - 实现了 3D 结构数据的持久化存储，确保历史记录中能正确回显 3D 模型。

## [2.1.0] - 2025-12-23

### 新增
- 🤖 **Agentic NMR 分析**
  - 引入代理工作流，结合 VLM (视觉大模型) 进行峰值提取
  - 使用 LLM 进行逻辑推理，RDKit 进行化学规则验证
  - 实现高精度的核磁共振谱图解析
- 📚 **RAG 知识增强**
  - 支持上传 PDF/TXT 等化学文献构建本地知识库
  - 基于 ChromaDB 向量检索，让回答有据可依
- 💾 **持久化历史记录**
  - 基于 SQLite 的完整对话历史存储
  - 支持图片上传回显和历史会话管理
- 🧬 **实时结构可视化增强**
  - 自动识别对话中的化学物质
  - 实时生成 2D 分子结构图并持久化存储为静态文件

### 改进
- 🔧 **后端架构优化**
  - 统一使用 SiliconFlow API (GLM-4.6V) 替代原有的多模型方案
  - 优化了文件上传和静态资源服务的配置
- 🐛 **Bug修复**
  - 修复了刷新页面后上传图片在历史记录中消失的问题
  - 修复了结构图生成失败的问题

### 主要功能模块

#### API 端点
- `POST /api/chat/send` - 发送消息
- `GET /api/chat/history/{conversation_id}` - 获取对话历史
- `GET /api/chat/conversations` - 列出所有对话
- `DELETE /api/chat/conversations/{conversation_id}` - 删除对话
- `POST /api/image/upload` - 上传图片
- `POST /api/image/analyze` - 分析图片
- `GET /api/image/{image_id}` - 获取图片信息
- `DELETE /api/image/{image_id}` - 删除图片
- `GET /api/health` - 健康检查
- `GET /api/health/ping` - 简单 ping

#### 前端页面
- `+page.svelte` - 首页组件
- `spectrum/+page.svelte` - 光谱分析页面
- `api-docs/+page.svelte` - API文档页面
- `app.html` - 应用模板
- `app.css` - 全局样式

#### 核心服务
- `RAGService` - 检索增强生成服务
- `LLMService` - 大语言模型服务
- `ImageService` - 图像处理服务

## 版本说明

### 版本号格式
本项目使用语义化版本号 `MAJOR.MINOR.PATCH`：

- **MAJOR**: 不兼容的 API 更改
- **MINOR**: 向后兼容的功能添加
- **PATCH**: 向后兼容的错误修复

### 更改类型
- **新增**: 新功能
- **更改**: 现有功能的更改
- **弃用**: 即将移除的功能
- **移除**: 已移除的功能
- **修复**: 错误修复
- **安全**: 安全相关的修复

### 发布周期
- **主要版本**: 根据需要发布
- **次要版本**: 每月发布
- **补丁版本**: 根据需要发布
